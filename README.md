# 📊 Blinkit Sales Analysis Dashboard

This project is an interactive Power BI dashboard created to analyze Blinkit grocery sales data and generate meaningful business insights.

## 🔍 Project Overview
The dashboard provides a comprehensive analysis of sales performance across different outlets, item categories, and locations using dynamic visualizations.

## 📌 Key Insights
- Total sales and average sales analysis
- Category-wise product performance
- Outlet size and location comparison
- Sales trends based on outlet establishment year
- Customer rating and item visibility analysis

## 🛠 Tools Used
- Microsoft Power BI
- Data Cleaning & Transformation
- Basic DAX Measures
- Data Visualization

## 📈 Key Features
- Interactive filters and slicers
- KPI cards for quick insights
- Business-focused dashboard design
- User-friendly layout for easy understanding

## 🎯 Objective
To transform raw retail data into actionable insights and strengthen practical skills in data analytics and business intelligence.
